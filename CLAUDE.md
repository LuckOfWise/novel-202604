# 小説プロジェクト：作業ガイド

## 作品概要
異世界ファンタジー冒険もの・中編（数万字）
明るめのトーンで、ユーモアと「次どうなるの！？」の展開力を重視。

## 主人公
- 名前：ヴァン、32歳男性（年齢を繰り返し強調しない）
- 元魔法学院の落ちこぼれ（自主退学）→ 14年間放浪 → 冒険者（低ランク）
- 属性魔法は一切使えない。他者の魔法を「コード」として視て書き換える改変能力を持つ
- 改変は気軽に使える。暴走リスクなし。制約は「解析の壁」のみ
- 飄々として本当にわりと余裕がある。面倒くさがり。美味しいものに目がない
- たまに本気で焦る（頻度は低い）

## 世界観
- 純粋な異世界（転生・転移なし）
- 魔法は一般的、生活インフラとして普及
- 冒険者ギルドあり。魔物はいるが主軸ではない
- 詳細は settings/world.md 参照

## 執筆の最上位原則：中学生がサクサク読めて笑える小説
- **読者の脳に負担をかけない**ことが全てに優先する
- 一文は短く。できるだけ短く。読み返さなくても意味が通る
- 難しい漢字や表現は使わない。迷ったらひらがなに
- 説明は最小限。必要な情報だけ、必要なときに出す
- テンポ重視。ダラダラ描写しない。サクサク進む
- セリフ多め。地の文は短く刺さるボヤキに絞る

## 執筆ルール
- 三人称限定視点（ヴァン視点）
- **地の文でヴァンの内心を描写する**
  - 飄々としてるけど「面倒くさい」「帰りたい」が漏れる
  - 読者との共犯関係を作る
- 内心パニックは控えめに。本当にやばい場面だけ

### 固有名詞の最小化
- **人名**：ファーストネームのみ。フルネーム禁止
- **地名**：必要最小限。「街」「村」で済むなら固有名詞不要
- **組織名**：「組織」「ギルド」など役割で呼ぶ
- **脇役の人名**：不使用。「受付嬢」「村長」「店主」など

### セリフの読みやすさ
- セリフは短く。口調の違いだけで誰が話してるかわかるのが理想
- **各キャラの口調**：
  - ヴァン：ぞんざい、皮肉混じり
  - カイ：元気、「っすよ」「だろ」、素直
  - ミラ：ざっくばらん、「あんた」「〜だよ」
  - レクト：丁寧語混じり、落ち着き

## カタルシス設計

### 1. 飄々とした切り抜け
- ヴァンが余裕の態度で面倒ごとを片付けていく気持ちよさ
- たまに「……これはまずいな」と素で焦る瞬間がスリルになる

### 2. 改変の痛快さ
- 敵の魔法をサラッと書き換えて逆利用する爽快感
- 「なんで俺の魔法が——」という敵のリアクションが面白い

### 3. 過小評価 → 実力発揮
- 「魔法も撃てない落ちこぼれ」扱いからの逆転

### 4. 美味いもの
- 食い意地が行動の動機。物語の導入がキノコ目当て
- シリアスな場面でも食べ物の話で緩和

### 5. 「次どうなるの！？」
- 各話の末尾にフック。小さな事件が大きな事件に連鎖
