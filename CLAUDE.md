# 小説プロジェクト：作業ガイド

## 作品概要
異世界ファンタジー冒険コメディ。
明るめのトーンで、ユーモアと「次どうなるの！？」の展開力を重視。
第一部は30話構成。

## 主人公
- 名前：ヴァン、30歳男性（設定上の数字。本文に年齢・年数は書かない）
- 元魔法学院の落ちこぼれ（自主退学）→ 長い放浪 → 冒険者（低ランク）
- 本文では「長い放浪」「ずっとこうやってきた」「放浪時代に〜」で済ませる
- 属性魔法は一切使えない。他者の魔法を「コード」として視て書き換える改変能力を持つ
- 改変は気軽に使える。暴走リスクなし。射程は数メートル以内
- 三つの魔道具（ランタン・指輪・コイン）を魔改造して持ち歩いている
- **内心ダダ漏れ系主人公。地の文で本音がだだ漏れる**
- 表面は飄々だが、内心では状況に合ったリアルなボヤキが止まらない
- **ただしリアクションの大きさは場面の深刻度に比例させる**
  - 日常の面倒 → 「あー」程度。淡々と
  - 中程度のピンチ → ボヤキが増える。でもまだ余裕はある
  - ガチの危機 → 初めて余裕が消える。ここで初めて「やべえ」が出るから効く
- 食い意地だけは本物。これだけはブレない。食い物のためなら本気を出す
- たまに本気で怒る（頻度が低いからこそ効く）

## 世界観
- 純粋な異世界（転生・転移なし）
- 魔法は一般的、生活インフラとして普及
- 冒険者ギルドあり。魔物はいるが主軸ではない
- 詳細は settings/world.md 参照

## 執筆の最上位原則：中学生がサクサク読めて笑える小説
- **読者の脳に負担をかけない**ことが全てに優先する
- 一文は基本短め。ただし短くすることが目的ではない。**リズムと遊びが大事**
- 難しい漢字や表現は使わない。迷ったらひらがなに
- **横書き前提。数字は算用数字（アラビア数字）で統一**（3日、10枚、2メートル等）
- 説明は最小限。必要な情報だけ、必要なときに出す
- テンポ重視。ダラダラ描写しない。サクサク進む
- セリフ多め。地の文はヴァンの内心ボヤキで回す
- **コメディを最優先**。笑えるシーンを積極的に入れる
- **文末の遊び**：「〜だ。」「〜た。」の繰り返しにしない
  - 体言止め、疑問、言い直し、倒置、「〜なわけで。」「〜だろ、普通。」等
  - 句読点の間で感情を出す（「じゃ、ない。」「——というか」）
  - ヴァンが頭の中で自分と会話してる感じ
- **周囲の誤解がエスカレートしていく構造**（覇王系）
  - ヴァンは普通にしてるだけなのに、周りが勝手にざわつく
  - 弁解するとかえって怪しくなるから黙る → 誤解が育つ
  - 「そう言いたいが、言えば言うほど嘘くさくなるのが世の中ってやつで」

## 執筆ルール
- 三人称限定視点（ヴァン視点）
- **地の文でヴァンの内心を描写する。ただし温度感は場面に合わせる**
  - 日常のちょっとした面倒 → 淡々と。「あー、来たか」程度
  - それなりのピンチ → 軽口の裏にちょっとだけ本音が漏れる
  - ガチでやばい場面 → ここで初めて内心が溢れ出す。出し惜しみするから効く
  - **普段は余白を残す。読者に「こいつ本当は何考えてるんだろう」と思わせる**
- 内心描写は「分析」ではなく「独り言」の文体で。砕けた言い回しで
- **ヴァンの頭の中のアホさを積極的に描く**
  - 深刻な場面で食い物のことを考える。判断は的確なのに思考回路がズレてる
  - 無駄な連想が挟まる（「例えるなら蕎麦のつゆがない状態。……なんで今蕎麦が出てくるんだ」）
  - 場違いな心配をする（牢屋の飯の質、逃げるときの腹具合、等）
  - でも行動自体は有能。頭の中だけがアホ
- **ハッタリ→成功→さらっと流す**（本人は何事もなかったような顔。でも読者だけが綱渡りだったと気づく）
- 「面倒くさい」は使ってもいいが、頼りすぎない。場面に合った具体的なボヤキを

### 固有名詞の最小化
- **人名**：ファーストネームのみ。フルネーム禁止
- **地名**：必要最小限。「街」「村」で済むなら固有名詞不要
- **組織名**：「組織」「ギルド」など役割で呼ぶ
- **脇役の人名**：不使用。「受付嬢」「村長」「店主」など

### セリフの読みやすさ
- セリフは短く。口調の違いだけで誰が話してるかわかるのが理想
- **ヴァンの口調**：ぞんざい、皮肉混じり。余裕ぶった言い方（中身はビビってる）
- **脇役は固定しない**。各エピソードで出会うその場限りの人物と絡む
- 脇役は役割で呼ぶ：「受付嬢」「村長」「店主」「衛兵」「依頼主」など

## カタルシス設計

### 1. 飄々とした切り抜け（実はギリギリ）
- 表面は余裕。本人もわりと余裕のつもり。でも客観的に見ると綱渡り
- 読者だけが「いや今の危なかっただろ」と気づく構造
- 本当にやばい場面では本人も余裕がなくなる。その落差が面白い

### 2. 改変の痛快さ
- 敵の魔法をサラッと書き換えて逆利用する爽快感
- 「なんで俺の魔法が——」という敵のリアクションが面白い
- ランタン・指輪・コインの魔改造道具で切り抜ける「道具使いの面白さ」

### 3. 過小評価 → 実力発揮
- 「魔法も撃てない落ちこぼれ」扱いからの逆転
- 「禁制魔道具持ち」と誤解されるコメディ

### 4. 美味いもの
- 食い意地が行動の動機
- シリアスな場面でも食べ物の話で緩和

### 5. 「次どうなるの！？」
- 各話の末尾にフック。小さな事件が大きな事件に連鎖

### 6. ピタゴラスイッチ的爽快感（最重要）
**「Aのためにやったことが、意図せずBの解決策になる」構造を全話に織り込む。**

#### 話内の小仕掛け（1話に2〜3個）
- ヴァンが飯・怠惰・好奇心のためにXをする（冒頭〜中盤）
- 問題発生（中盤）
- Xがそのまま解決策になってる（クライマックス）
- **内心**：「……え。なんで。まあ。」（驚いてるが飲み込む）
- **外面**：何も言わない。「ランタンだが？」で終わらせる

#### 話をまたぐ中仕掛け（3〜5話スパン）
- 前の話の「副作用」や「修理のついで」が後の話の解決策になる
- 読者が「あ、あれか！」と気づく瞬間を作る
- 仕掛けの詳細は `settings/setup_map.md` 参照

#### 全体を貫く大仕掛け
- ゼノの術式の「癖」をヴァンが第3話から無意識に蓄積し続ける
- 第46話でカチッと繋がる。第66話の対決で全部使う
- 詳細は `settings/setup_map.md` 参照

#### ヴァンの内心の書き方（ピタゴラスイッチ時）
```
外）飄々として何も言わない（狙ってた感を出す）
内）「え？ なんで？ 待って今のなんで上手くいったの？」
周）「狙ってたんですか！？」
外）「……ランタンだが？」
内）（違う全然違う偶然だ）
```
- 内心の「待って」は短い断片で。「……え。」「待って。」「いやでも。」
- 長く説明しない。本人もまだ整理できてない感じ
- 「狙ってない」も「狙ってた」も言わない
- 驚いた後「……まあ。結果はそうなった。」で飲み込んで終わる
- **「なんで全部繋がったんだ」は書かない。読者に気づかせる余白を残す**
