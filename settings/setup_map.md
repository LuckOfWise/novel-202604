## 大仕掛け：ゼノの術式の「癖」

**コンセプト**：ヴァンは気づかないまま、全話通じて同じ人物（ゼノ）の魔法を読み続けてきた。

| 話 | 内容 | 種別 |
|----|------|------|
| 第3話 | 転送陣の部品を解析。「独学の書き方だ。荒いけど才能がある」と思う（蕎麦食いながら） | **植える** |
| 第12話 | 農地の暴走魔法を読む。「……また同じ癖。同じやつが書いてる」（でも追う気はない） | **蓄積** |
| 第18話 | 倉庫の術式。「3回目だ。この荒い書き方」（煩わしそうに。固有名詞なし） | **蓄積** |
| 第29話 | 山間の実験跡の残骸。「またか。このやつ、あちこちで暴れてるな」 | **蓄積** |
| 第38話 | 鉱山の研究所の術式。「全部同じだ。ゼノ、だったか。全部こいつが書いてたのか」（初めて繋がる） | **回収①** |
| 第46話 | 「ゼノという名前」を聞いた瞬間、既視感。「……ああ、そうか。あの癖の」 | **回収②** |
| 第57話 | 晩餐でゼノの魔法を間近で見る。「お前の書き方、ずっと見てきた」（口には出さない） | **回収③** |
| 第66話 | 対決。ゼノの術式が全部読める。「放浪中からずっと読んでたからな」（内心のみ） | **大回収** |

**書き方の注意**：
- 第3〜29話では「同じ癖」「荒い書き方」「才能がある」という表現を一貫させる
- 人物名は出さない。ヴァンは誰が書いたか気にしない（飯に集中してるから）
- 第38話で「ゼノ」という名前に触れるが、それ以前の癖と繋げるのは内心のみ

## 中仕掛け⑤：道具の相互補完パターン（各話で使う）

**コイン → 指輪の切り替え**（射程問題）
- コインが弾かれる・届かない → 指輪で直接触れに行く
- 初出第23話。以降の戦闘でも使う

**ランタン → コインの射程補助**
- ランタンで状況を読んで、コインを精確に投げる
- ランタンなしだと「だいたい」の感覚で投げるしかない

**指輪の微調整が後で機能する**
- 修理でいじった設定が「たまたま」戦闘に有効な状態になってる

## 話内の小仕掛けパターン集（毎話2〜3個使う）

### パターンA：飯のためにした行動が解決策
例）食堂の魚の鮮度を確かめるため保存魔法を調べた → その術式が敵の魔法と同じ構造

### パターンB：修理のついでが防衛になる
例）「どうせなら出力上げとくか」という雑な判断 → 想定の3倍の効果が出る

### パターンC：断った依頼の内容がヒントになる
例）断った護衛依頼の対象物を見てた → それが後の問題の原因だとわかる

### パターンD：もらった食い物に情報がついてくる
例）りんごをくれた八百屋のおっさんが「最近おかしな旅人が来た」と言ってた

### パターンE：ヴァンが「余計なことをしすぎた」改変が機能する
例）「3層で足りるのに5層まで書き換えた（念のため）」→ その2層が後で使われる

## 回収の書き方ルール

1. **内心の「……え。」は短く**。断片的に。整理できてない感じ
2. **外面では何も言わない**。「ランタンだが？」「街灯だが」で終わらせる
3. **「狙ってた」とも「狙ってない」とも言わない**
4. **「なんで全部繋がったんだ」は書かない**。読者に気づかせる
5. 飲み込みの定型：「……まあ。結果はそうなった。」「……考えるのはやめよう」
6. 周囲が「狙ってたんですか！？」と聞いてくるのが最大の爽快感
