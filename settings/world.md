# 世界観設定

## 世界の基本構造
- 純粋な異世界（転生・転移なし）
- 複数の国家・都市国家が並立する大陸
- 実力主義が浸透した社会。才覚と実績があれば身分を問わず評価される

## 種族
- 人間のみ

## 宗教・信仰
- 統一的な宗教は存在しない。各地に土着の信仰や精霊信仰が残る程度
- 戦争において宗教が大きな役割を果たすことはない

## 技術水準
- 中世後期〜近世相当。魔法が技術を一部代替している
- 印刷術はあるが普及途上。情報伝達は人の足と魔法通信に頼る
- 火薬は存在しない。攻城兵器は魔法と機械の複合型

## 魔法体系

### 基本原理
- 魔法は「魔素」と呼ばれる大気中のエネルギーを、体内の「魔核」を通して変換・行使する技術
- 魔核の容量（＝魔力量）は生まれつき決まっており、後天的にはほとんど増やせない
- 魔素は大気中に偏在する。山岳部や森林に多く、乾燥した平原にはやや少ない
- 魔法の行使には魔力を消耗する。消耗した魔力は休息と食事により回復するが、回復速度には個人差がある
- 魔力を使い果たすと「魔核枯渇」を起こし、意識喪失・最悪の場合は死に至る

### 二系統：直感型と分析型

**直感型**
- 才能と感覚で魔法を操る。イメージを魔力に直接変換するため、即応性と出力に優れる
- 訓練法：実戦と反復。「体で覚える」型。師匠の技を見て盗むのが伝統的
- 社会的評価：**高い**。目に見える成果（破壊力、速度、規模）が分かりやすく、軍でも高く評価される
- 弱点：効率が悪い。魔力の変換ロスが大きく、10の魔力で3〜5程度の効果しか出せないことが多い。本人はそれに気づかないことが多い
- 代表的使い手：カイ、マルクス

**分析型**
- 理論と観察に基づいて魔法を構築する。魔法の構造を分解・理解し、最適な経路で魔力を変換する
- 訓練法：座学と実験。魔法の理論書を読み、仕組みを理解したうえで実践する。地道で時間がかかる
- 社会的評価：**低い**。出力が低く見えるため「弱い魔法使い」と見なされがち。学者や技術者向きとされ、前線では軽視される
- 強み：効率が極めて高い。10の魔力で8〜9の効果を引き出せる。また、他者の魔法を分析して弱点を見抜くことが可能
- 代表的使い手：ヴァン、セレン（セレンは分析型の素養を軍事戦略に転用している）

**社会的評価の差が生む偏見**
- 大陸全体で直感型が「本物の魔法使い」とされる風潮が根強い
- 分析型は「頭でっかち」「実戦で使えない」「魔力が弱い者の言い訳」と揶揄される
- 軍の昇進でも直感型が圧倒的に有利。分析型は後方支援や研究職に回されることが多い
- この偏見が、ヴァンが「ただの荷運び崩れ」として軽視される背景の一つになっている
- 実際には分析型の応用力は戦略レベルで極めて有用だが、それを理解できる人間が少ない

### 魔法の具体的な使用例

**軍事用途**
- 攻撃魔法（火球、雷撃、風刃など）：直感型が主力。破壊力重視
- 防御・結界魔法：陣地防衛や行軍中の警戒網。分析型が得意とする数少ない「認められた」領域
- 魔法通信：短距離の音声伝達。長距離は専用の通信士が中継する。リアルタイム性に限界がある
- 偵察・探知魔法：敵軍の接近を感知する。精度は術者の技量に大きく依存
- 兵站魔法：食料の保存延長、水の浄化、軽量化の付与。地味だが軍の行動力を左右する

**交易・民間用途**
- 鑑定魔法：品物の真贋判定、品質確認。商人の必須技能
- 保存魔法：食料や薬品の劣化を遅らせる。交易路が長い大陸では死活的に重要
- 照明・暖房：魔法灯、魔法炉。都市部では普及しているが辺境では薪が主流
- 治癒魔法：傷の治療、病気の緩和。魔力消耗が大きいため高価。前線での治癒士は常に不足
- 通信魔法：商人間の短距離連絡。長距離は費用が高く、緊急時にしか使われない

**日常生活**
- 大多数の一般人は微弱な魔力しか持たず、日常の補助程度にしか使えない
- 魔法を本格的に使えるのは全人口の1〜2割程度。うち軍や商業で活躍できるレベルはさらに一握り
- 魔法が使えない人間が差別されることはないが、出世の天井が低くなる傾向はある

### 魔法の制約とコスト
- **魔力消耗**：全ての魔法は魔力を消耗する。消耗量は術の規模と精度に比例
- **回復に時間がかかる**：全力行使した場合、完全回復に丸1日以上。戦場では魔力管理が生死を分ける
- **集中力の限界**：複雑な魔法ほど集中力を要し、疲労・負傷・精神的動揺で精度が落ちる
- **相殺と干渉**：複数の魔法が同時に行使されると互いに干渉する。大規模戦闘では「魔法の渋滞」が発生し、個々の魔法の威力が落ちる
- **距離減衰**：魔法の効果は距離に応じて減衰する。遠距離攻撃は出力の大半が失われる
- **物質の抵抗**：石壁や金属は魔法を遮断・減衰させる。城壁が未だに有効な理由

### 魔法教育
- **グランデール軍学院**：大陸最大の魔法教育機関。直感型の戦闘魔法使い育成に特化。分析型の教育課程は形だけ存在するが予算も人材も薄い
- **ロッセの私塾群**：各都市に小規模な魔法私塾がある。商業向けの実用魔法（鑑定、保存、通信）を教える。体系的な理論教育はロッセが最も充実している
- **ヴェルドの口伝**：教育機関は存在しない。集落の長老や先輩から口伝と実践で学ぶ。自然環境と密接に結びついた独自の魔法体系を持つ
- **教育格差**：魔法教育を受けられるかどうかは出身地と経済力に大きく左右される。ヴァンは正規教育を受けていない。大陸中を放浪しながら独学で分析型の理論を築き上げた

## 三大勢力

### グランデール（軍事大国・東部）

**概要**
- 大陸東部の広大な平原地帯を支配する強国
- 皇帝を頂点とする中央集権制。軍功による昇進が制度化されている
- 現皇帝ヴォルフは拡張主義者。西方侵攻を推し進める

**主要都市**
- **帝都カルデア**：大陸東端に位置するグランデールの首都。皇帝の居城と軍学院がある。人口約15万。政治・軍事の中枢
- **軍港ベルガンド**：大陸東岸の港湾都市。海上交易と軍の兵站を担う。人口約8万。東方からの物資集積地
- **前線都市ラドム**：グランデール西端の軍事拠点。西方侵攻の前進基地として急速に軍事化された。元は交易で栄えた町だったが、現在は軍施設が町の大半を占める。人口約5万（うち軍人2万）

**軍事力**
- 総兵力：約12万。うち常備軍8万、動員兵4万
- 主力兵種：重装歩兵と魔法兵の混成部隊。正面からの力押しが基本戦術
- 魔法戦力：大陸最大。軍学院出身の魔法兵が組織的に運用される。直感型の大出力攻撃魔法が中心
- 西方方面軍：マルクス率いる3万の遠征軍。精鋭で構成され、これまでの西方侵攻の主力
- 弱点：兵站線が長い。東部からの補給路が延び切っており、交易路を押さえなければ遠征軍の維持が困難

**内部の派閥対立**
- **皇帝直轄派**（主戦派）：皇帝ヴォルフを中心に、領土拡張と交易路支配による覇権を目指す。セレンはこの派閥の実行部門を担う
- **旧貴族派**（穏健派）：伝統的な貴族層。これ以上の拡張は国力を消耗させると懸念。前線への増援を渋り、セレンと対立
- **軍人派**（現場主義）：マルクスに代表される前線指揮官たち。政治的野心はないが、兵の無駄死にに対する不満がくすぶる
- セレンは皇帝の後ろ盾で旧貴族派を抑えているが、戦争が長引けば反発が強まる

**弱点**
- 兵站の脆弱さ。長大な補給線を維持する負担が国庫を圧迫
- 内部の派閥対立。戦況が膠着すれば主戦派の求心力が低下
- 占領地の統治。征服した地域の反発を抑えるために兵力が分散される
- 海からの脅威への対処が薄い

### ロッセ商業連合（都市国家同盟・中央部）

**概要**
- 大陸中央部の交易路沿いに点在する都市国家の同盟
- 各都市の代表者による合議制。盟主が取りまとめるが拘束力は弱い
- 経済力は三勢力中最大だが、軍事力は最も脆弱

**主要都市**
- **盟主都市メルヴィス**：ロッセ最大の都市。商業連合の中枢で、盟主アルベルトの本拠地。大陸の東西交易路と南北交易路が交差する要衝。人口約12万。巨大な市場と取引所が都市の中心
- **港湾都市レーゲン**：大陸南岸の港。海上交易の拠点で、ロッセの対外窓口。人口約7万。傭兵団の多くがここを拠点にしている
- **学術都市コルネス**：交易路の北側に位置する知の都。魔法理論の研究と私塾が集中している。人口約4万。大陸で最も分析型魔法の研究が進んでいる
- **交易町ヘルダ**：メルヴィスの西方に位置する中規模の交易町。ガルドが隠居している町。現在は前線に近く、戦火に脅かされている。人口約1万5千

**軍事力**
- 常備軍：ほぼ存在しない。各都市の市民兵合わせて約1万
- 傭兵戦力：ヴィオラの灰鉄旅団（約300名）を含む複数の傭兵団を雇用。総数約5千
- 主力兵種：傭兵による機動防衛と、都市城壁を利用した籠城戦
- 魔法戦力：組織的な魔法部隊はない。個々の傭兵や商人が持つ魔法に依存
- 弱点：統一された指揮系統がない。各都市が自分の防衛を優先し、連合としての軍事行動が取れない

**内部の派閥対立**
- **抗戦派**：メルヴィスを中心に、グランデールへの抵抗を主張。アルベルトはこの立場だが、実行手段に乏しい
- **降伏派**：東部の都市群。グランデールに恭順することで交易の継続を図ろうとする。「戦っても勝てない」が論拠
- **中立派**：南部の港湾都市群。海上交易で食べていけるため、大陸内の戦争に深入りしたくない
- 合議制のため意思決定が遅く、緊急時の対応が致命的に遅れる

**弱点**
- 軍事力の決定的な不足。金で傭兵を雇う以外に戦力がない
- 合議制による意思決定の遅さ。降伏派の妨害で軍事予算すら通りにくい
- 交易路が分断されれば経済の生命線が断たれる
- 都市間の結束が弱く、個別に切り崩される危険

### ヴェルド辺境同盟（小国・部族連合・西部）

**概要**
- 大陸西部の山岳・森林地帯に暮らす小国や部族の緩い連合
- 各集落・部族の長が独自に統治。連合としての統一指揮系統はほぼない
- 個々の戦士の質は高いが、組織戦が苦手

**主要都市・拠点**
- **砦都市ゲルド**：ヴェルド最大の拠点。山間の盆地に築かれた要塞都市。部族長たちの合議がここで行われる。人口約2万。グランデールの侵攻に対する防衛の中心
- **森の砦カレン**：深い森林の中に築かれた軍事拠点。ゲルドの東方前線を支える防衛拠点。ゲリラ戦の出撃基地。常駐兵力約2千
- **山岳集落ノール**：リーナの出身地近くの集落群。標高が高く攻めにくいが、冬季は孤立する。人口約3千

**軍事力**
- 総兵力：約2万。ただし正規軍ではなく、各部族の戦士の集合体
- 主力兵種：山岳・森林でのゲリラ戦。地形を利用した奇襲、伏撃、遊撃
- 魔法戦力：組織的な運用はないが、個々の戦士が自然環境と連動した独自の魔法を使う。カイのような突出した魔力の持ち主も散在
- 弱点：平地での正面戦闘に弱い。組織的な連携が苦手。補給体制が貧弱で長期戦に耐えられない

**内部の派閥対立**
- **徹底抗戦派**：西奥の山岳部族。土地を捨てることは死を意味するため、最後まで戦うと主張。カイはこの立場
- **撤退派**：東部の部族。前線を放棄し、山岳の奥地に退いて持久戦を提案。犠牲を減らしたい
- **ロッセ依存派**：一部の長が、ロッセの経済力を頼って傭兵を雇い防衛を委託しようとする
- そもそも統一された意思決定機関がないため、部族ごとにバラバラに動いてしまう

**弱点**
- 統一指揮の欠如。部族間の連携が取れず、各個撃破されやすい
- 補給・兵站の脆弱さ。長期戦になると食料・武器が尽きる
- 平地での戦闘能力の低さ。山を出れば圧倒的に不利
- 外交力の不足。他勢力との交渉に不慣れ

## 戦争の具体的な状況

### 時系列
- **2年前**：グランデール皇帝ヴォルフが西方侵攻を宣言。「交易路の安全確保」を大義名分とする
- **1年半前**：グランデール西方方面軍（マルクス指揮）がロッセ東部に進軍。ロッセ東端の交易拠点3都市が陥落
- **1年前**：セレンが参謀総長に就任し、戦略を再編。ヴェルド辺境への圧力を強化。ヴェルド東部の集落群が陥落
- **8ヶ月前**：リーナの所属していた斥候部隊がロッセとの交渉の犠牲にされ壊滅
- **6ヶ月前**：グランデール軍がヴェルドの森の砦カレンに迫る。カイを含むヴェルド戦士たちが必死の防衛戦
- **3ヶ月前**：ロッセのメルヴィスでグランデールとの講和交渉が決裂。アルベルトが傭兵団の大規模雇用を決断
- **現在**（物語開始時点）：膠着状態。グランデールは兵站の限界で攻勢を一時停止。セレンが次の一手を準備中

### 現在の前線

**東部前線（グランデール―ロッセ間）**
- グランデール軍はロッセ東端まで進出。占領した3都市を兵站基地として使用
- 前線はメルヴィスの東方約150キロの丘陵地帯「エルバの丘」で膠着
- ヴィオラの灰鉄旅団がこの前線の一角を担当

**西部前線（グランデール―ヴェルド間）**
- グランデール軍の別働隊がヴェルドの森林地帯に侵入
- 森の砦カレンの東方で攻防が続く。ヴェルドのゲリラ戦が奏功し、進軍は遅い
- しかし補給が続く限り、グランデールの物量にヴェルドは耐えきれない

**交易町ヘルダ周辺**
- ガルドが暮らす交易町。メルヴィスの西方に位置し、両前線の間に挟まれた危険地帯
- グランデール軍の偵察隊が出没し、物資の略奪が頻発
- 住民の避難が始まっているが、ガルドは人脈を頼りに留まっている

### これまでの主要な戦い

1. **エルバの丘の戦い**（1年前）：グランデール西方方面軍とロッセ傭兵連合の最初の大規模衝突。マルクスの直感型魔法が戦場を蹂躙し、ロッセ側は大損害を受けた。ただしロッセ側の粘り強い籠城戦で完全な突破は阻止。以降、この地で膠着が続く
2. **カレンの森の攻防**（6ヶ月前〜現在）：グランデールの別働隊がヴェルドの防衛拠点「森の砦カレン」を目指して侵攻。ヴェルド側は森林を利用したゲリラ戦で遅滞させているが、じわじわと押されている。カイの魔力が唯一の切り札だが、消耗が激しい
3. **ヘルダ街道の小競り合い**（3ヶ月前〜）：交易町ヘルダ周辺でグランデールの偵察隊とロッセの護衛隊が散発的に衝突。大規模戦闘にはなっていないが、交易路の安全が脅かされている

### 各勢力の現在の戦略方針

- **グランデール（セレン）**：正面からの力押しは兵站の限界で一時停止。次の一手として、ロッセ内部の降伏派を扇動し、ヴェルドとロッセの分断を狙う。同時に交易路の要衝を個別に確保して経済的に締め上げる戦略
- **ロッセ（アルベルト）**：外交で時間を稼ぎつつ、ヴェルドとの軍事協力を模索。しかし降伏派の妨害と傭兵の高額な維持費に苦しんでいる
- **ヴェルド**：山と森に頼ったゲリラ戦で時間を稼ぐ。しかし統一指揮がないため戦線が散発的で、補給も底を突きつつある

### 民間人への影響
- **難民**：ヴェルド東部とロッセ東端から大量の難民が発生。メルヴィスや内陸都市に流入し、社会不安の原因に
- **物価高騰**：交易路の分断で物資が滞り、食料・武器・魔法触媒の価格が2〜3倍に。特にヴェルドでは深刻な食料不足
- **治安悪化**：戦争で職を失った人々が盗賊化。前線から離れた地域でも略奪事件が増加
- **情報の混乱**：各勢力の宣伝と偽情報が入り乱れ、何が真実か判別しにくい状況。エリスのような情報屋の価値が急上昇
- **傭兵需要の急増**：ロッセを中心に傭兵の需要が爆発。質の低い傭兵団も乱立し、略奪まがいの行為も

## 交易路の詳細

### 大街道（東西大動脈）
- **ルート**：帝都カルデア → ベルガンド → ラドム → （エルバの丘）→ メルヴィス → ヘルダ → ゲルド方面
- 大陸を東西に貫く最大の交易路。幅広の街道で馬車が双方向に通行可能
- **主要交易品**：東→西：穀物、鉄鉱石、武器、織物。西→東：薬草、木材、毛皮、鉱石（希少金属）
- **現状**：エルバの丘で前線が形成されており、東西の直通は事実上不可能。メルヴィス以東はグランデールの支配下。ヘルダ周辺は危険地帯

### 南方海路
- **ルート**：ベルガンド（東岸）→ 海上 → レーゲン（南岸）
- 海路で東西を迂回する代替ルート。大街道が封鎖された現在、ロッセの生命線
- **主要交易品**：グランデールからの穀物（非公式に取引される）、南方からの香辛料・絹
- **現状**：航行可能だが海賊の増加と保険料の高騰で採算が悪化。グランデールが海上封鎖を検討中との噂

### 北方山道
- **ルート**：コルネス → 山間の峠道 → ノール方面 → ゲルド
- ロッセ北部からヴェルド西部へ抜ける山間の道。狭く険しいが、前線を迂回できる
- **主要交易品**：ロッセからの食料・日用品。ヴェルドからの薬草・希少鉱石
- **現状**：細い道なので大量輸送には向かない。しかしヴェルドへの唯一の安全な補給路として重要性が急上昇。盗賊の出没が問題

### ヘルダ街道
- **ルート**：メルヴィス → ヘルダ → 森の砦カレン方面
- 大街道から分岐してヴェルドの東部に至る道。ヴェルドとロッセを結ぶ陸路の要
- **主要交易品**：平時はヴェルドの木材と薬草がロッセに流れ、ロッセの加工品がヴェルドに入る主要ルートだった
- **現状**：グランデールの偵察隊が跋扈し、極めて危険。護衛なしでは通行不可能。ガルドのいるヘルダはこの街道の中間地点

### 交易路分断の影響
- 大街道の封鎖により、大陸の東西交易がほぼ停止。ロッセの経済は深刻な打撃を受けている
- 南方海路への依存が高まるが、輸送量に限界があり、物価高騰の根本的解決にならない
- ヴェルドへの補給は北方山道に頼るしかないが、冬季は積雪で通行不能になる（これは物語の時間的プレッシャーになる）
- 交易路を誰が支配するかが戦争の帰趨を決める。セレンが狙っているのもまさにこの点

## 地理の詳細

### 大陸全体図（概念）
```
          北
    ┌──────────────┐
    │   コルネス     │
    │  ノール ゲルド  │  ← ヴェルド（山岳・森林）
    │   カレン       │
    │ ヘルダ         │
    │  メルヴィス    │  ← ロッセ（交易路都市群）
    │         ラドム │
    │ レーゲン       │  ← 南岸港湾
    │     ベルガンド  │
    │     カルデア   │  ← グランデール（平原）
    └──────────────┘
          南
    ← 西         東 →
```

### 主要地形
- **グランデール領**：広大な平原と穀倉地帯。大規模農業と牧畜が盛ん。軍の大部隊が展開しやすい地形
- **ロッセ領**：交易路沿いに発展した都市群。丘陵地帯が多く、都市は天然の防御に恵まれた場所に築かれている
- **ヴェルド領**：険しい山岳地帯と深い森林。通行困難な地形が天然の要塞。ただし農地が少なく、食料自給が難しい
- **エルバの丘**：ロッセ東部の丘陵地帯。現在の東部前線。起伏が多く、攻城戦・陣地戦の舞台
- **カレンの森**：ヴェルド東部の深い森林地帯。森の砦カレンを中心とした防衛線。視界が悪く、ゲリラ戦に向く
- **ヘルダ盆地**：ヘルダ街道沿いの小さな盆地。交易町ヘルダがある。丘に囲まれ防衛しやすいが、包囲されると逃げ場がない

### 物語の主要舞台

**第一幕の舞台：ヘルダ周辺**
- ヴァンがガルドを救うために向かう場所。交易路の要衝であり、両前線の間の危険地帯
- 小さな町だが商人が集まるため情報が交錯する。ヴァンの情報収集に適した環境

**第二幕の舞台：メルヴィス**
- ロッセの中枢都市。アルベルトとの接触、ヴィオラの傭兵団との合流が起こる場所
- 政治的な駆け引きの舞台でもある

**第三幕の舞台：エルバの丘〜カレンの森**
- 実際の戦闘が行われる前線。ヴァンの作戦が戦局を動かす舞台
- カイとの合流、マルクスとの間接的な対峙が起こる

## 通貨と経済

- **ロッセ・クローネ**：ロッセ商業連合の発行する銀貨。大陸で最も広く流通する商業通貨
- **グランデール・マルク**：グランデールの金貨。高額取引に使用。軍の給与もこれで支払われる
- **ヴェルドの物々交換**：ヴェルドでは通貨よりも物々交換が主流。薬草や毛皮が交換の基準
- 戦争により通貨の信用が揺らぎ、物々交換に逆戻りする地域が増加
